# Changelog

## [1.0.0] — 2026-06-21

### Added
- Landing page with auto-rotating product gallery
- Product catalog with roast-level filtering, search, and category tabs
- Product detail page with grind selector, add-to-cart, and related products
- Shopping cart with quantity controls, coupon, and shipping calculation
- Checkout flow with shipping method, payment options, and order confirmation
- Admin dashboard with revenue stats, order table, and stock overview
- Customer auth pages (login & register) with validation
- About page with company story, timeline, and team values
- Contact page with inquiry form and business info
- Dark mode support with system preference detection
- Responsive layout — mobile-first, tablet, desktop
- Custom 404, loading, and error pages
- Design system tokens (Olive + Brick + Paper palette)

### Technical
- Next.js 16 with App Router and Turbopack
- TypeScript strict mode throughout
- Motion (motion/react) for animation
- Cart state via useSyncExternalStore + localStorage
- Tailwind CSS v4 design tokens
- Neobrutalist design with hard shadows and 2px borders
